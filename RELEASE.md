# Release v1.0.0

## Quick Start

```bash
pip install -r requirements.txt
python app/gesture_3d_main.py
```

## Destaques

- Controle 3D por gestos com MediaPipe + OpenGL.
- Suavizacao de movimento e deadzone.
- Modo avancado com duas maos.
- Overlay com FPS e status em tempo real.

## Notas

- Para escolher a webcam correta, defina `GESTURE_CAMERA_INDEX`.
- Adicione `assets/demo.gif` e `assets/screenshot.png` para enriquecer o README.
